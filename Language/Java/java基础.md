### JDK

### 命名
> package 规范

    例如 com.icss.pet.ui
    范围.公司.项目名.层
    全小写
    
> class 规范

    首字母大写
    
> 对象命令
    
    首字母小写
    例如
    User user = new User();
    
### 实体类
> entity 只有属性，没有方法
> POJO (plain old java object)简单java类
> VO (value object)
> DTO (data transfer object)

作用是夸层进行数据传输

### 方法重载
> 方法同名不同参（相当于PHP默认参数）
